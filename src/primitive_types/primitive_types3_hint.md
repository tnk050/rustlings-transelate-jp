# primitive_types3 hint

決まったサイズの配列を作成するショートハンドを使えば 100 個の要素をタイプする必要はありません(もしお望みならその方法でも確実に作成できますが!)。

例としてこのようなのがあります:

`let array = ["まだいる?"; 10];`

ボーナス:`a.len() >= 100`で`true`を返すには他にどのようなやり方がありますか?
