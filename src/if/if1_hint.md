# if1 hint

このエクササイズはあなたがその気なら 1 行で解けます!

別の言語でも似たような書き方があります:

- C(++) の場合: `a > b ? a : b`
- Python の場合: `a if a > b else b`

Rust の場合で覚えていてほしいのは以下です:

- `if`の条件式はカッコで囲む必要が有りません。
- `if`/`else`は式です。
- それぞれの`if`/`else`ブロックは`{}`で囲みます。
